# **App Name**: AdventureNav

## Core Features:

- Interactive Map: Display a stylized world map with interactive landmarks using react-zoom-pan-pinch for a smooth drag and zoom experience.
- Landmark Navigation: Navigate to different sections of the website when a user clicks a landmark, triggering a smooth zoom-in transition.
- HUD Overlay: Fixed UI overlay with a 'Quest Log' (site menu) in the top-left and a quick-access dock at the bottom-center.
- Tooltip Hover States: Show a tooltip with the section name and a short description when hovering over a landmark.
- Parallax Effect: Implement a subtle mouse-follow parallax effect to create a sense of depth on the map.
- Floating Markers: Use Framer Motion to add a subtle 'floating' animation to the markers.

## Style Guidelines:

- Primary color: Sunny yellow (#FFD700) to evoke a sense of adventure and vibrancy.
- Background color: Soft, desaturated green (#D3E4CD) for a natural landscape feel.
- Accent color: Sky blue (#87CEEB) for highlights and interactive elements.
- Body and headline font: 'PT Sans' for a modern, clean feel that is easy to read.
- Use simple, hand-drawn style icons for landmarks and UI elements.
- Full-screen layout with the map taking up most of the space, ensuring landmarks are easily visible and clickable.
- Smooth zoom-in transitions when navigating to different sections and subtle floating animations for the map markers.
